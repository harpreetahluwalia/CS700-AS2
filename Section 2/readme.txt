FUNCTIONALITY OF THE PROGRAM : 

Program will perform white-box test for a program that computes the sine and cosine functions in a specialized manner. 
It will test the function sin(int x) and cos(int x) functions from sin_cos.cpp file through various test cases. 
In sin(int x) function we have multiple conditional statements thus we perform test cases accordingly and test all the branches and statements of these function.

*********************************************************************************************************************************************

*********************************************************************************************************************************************
FILES OF THE PROGRAM (INCLUDING HEADERS AND IMPLEMENTATION) :

main.cpp			It is the main program file
test.h				It includes declaration of test() method which calculates actual output and matches with expected outputs for test case and prints the results.
test.cpp			It includes implementation of test() method which calculates actual output and matches with expected outputs for test case and prints the results.
sin_cos.cpp			It includes implementation of sin(int x) and cos(int x) functions.
